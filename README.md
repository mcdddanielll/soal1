# soal1
Pengguna memasukkan jarak tujuan → jika ≤10 km biaya 4.250, kalau lebih 6.000.


Lalu memasukkan panjang, lebar, dan tinggi paket → program menghitung volume = panjang × lebar × tinggi.


Jika volume >100, muncul pesan bahwa ada biaya tambahan 50.000.


Terakhir, program menampilkan total pengiriman (meskipun pada kode asli totalnya belum dihitung benar).
